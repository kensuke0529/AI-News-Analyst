AI News Analyst

Built by Kensuke Umakoshi | AI/ML Engineer & Data Scientist

An intelligent news analysis system combining Retrieval-Augmented Generation (RAG) with Wikipedia search. This project demonstrates LangGraph workflow orchestration, vector databases, and LLM deployment. The system routes queries between real-time news analysis and general knowledge retrieval, achieving 96% routing accuracy and 9.06/10 relevancy scores.
